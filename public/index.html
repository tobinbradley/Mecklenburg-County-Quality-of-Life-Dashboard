Created by Code for America and Lexington-Fayette Urban County Government • Source code on Github

  1. Welcome!

    The Quality of Life Dashboard is designed to help assess the health of neighborhoods in the City of Lexington.

  2. Metrics

    Use this control to select the Metric you're interested in. Metrics are divided into categories, and you can also search for Metrics using the search box.

  3. Map

    The Map is fully interactive - you can pan, zoom, and select neighborhoods by mouse or touch. If you zoom in close, the neighborhoods become transparent so you can view roads and map features.

  4. Search

    The Search control lets you search for almost anything. Type in a neighborhood number, the first part of an address or a place name, and select a match using your mouse, keyboard, or by touch.

  5. Year

    The Year controls let you control the year being shown on the map and in the Bar Chart. You can change the year by moving the slider and cycle the years available for the metric by using the Play/Pause button.

  6. Data Controls

    Use these links to clear any selected neighborhoods or view additional information including a table with District values, accuracy information, raw data, and weighted averages.

  7. Average

    The Average for Council Districts and any selected Districts is shown here. The Council District Average is indicated by a dashed line on the charts.

  8. Bar Chart

    The Bar Chart shows a lot of information about the Metric. First, it acts as a legend for the map - the map colors correspond to the the chart colors. It also shows you the density of the data - the taller the quantile, or bar, the more neighborhoods exist in that range. Clicking on a bar will select every neighborhood in that quantile.

    The dashed line represents the average value of the neighborhoods.

  9. Line Chart

    The Line Chart shows the trend of the Metric over time. The dashed line represents the average value of the neighborhoods.

  10. Metric Information

    The bottom of the page shows information about the Metric, including a brief description, why it is important, where the data came from, and additional resources related to the metric.

  11. Nav Bar

    The Nav Bar can show you more About the Quality of Life study, links to Download the data in a variety of formats, and Help you get started.
